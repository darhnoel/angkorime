Install KhmerIME for Windows

ដំឡើង KhmerIME សម្រាប់ Windows

Download

Use the unsigned x64 MSI installer.

Download MSI

Windows may show a warning because this build is unsigned.

Install

  1. Download the MSI.
  2. Run the installer as administrator.
  3. Choose KhmerIME in Windows input settings.

Uninstall

  1. Open Windows Settings.
  2. Find KhmerIME in Installed apps.
  3. Choose Uninstall.

Command line: msiexec /x @WINDOWS_FILE@

Install KhmerIME for Ubuntu

ដំឡើង KhmerIME សម្រាប់ Ubuntu

Download

Use the Debian package for Ubuntu and Debian-based systems.

Download .deb

Install

sudo apt install ./@LINUX_FILE@
ibus restart
  1. Open Ubuntu Settings.
  2. Go to Keyboard, then Input Sources.
  3. Add KhmerIME.

Uninstall

sudo apt remove khmerime
